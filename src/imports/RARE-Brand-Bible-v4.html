06

Photography & Imagery

✓ DO · Colour Temperature
  • Warm amber, honey, terracotta tones
  • Soft diffused light — never harsh flash
  • Cream, linen, and nude surfaces
  • Matte textures — stone, fabric, ceramic, wood
✓ DO · Subject Matter
  • Spa beds, towels, candles, steam
  • Hands applying skincare — close, intimate
  • Glass serums with warm reflections
  • Women at rest — eyes closed, peaceful
  • Botanicals, petals in cream/blush tones
✗ DON'T · Colour Temperature
  • Cool, blue, or grey-toned images
  • Bright clinical white backgrounds
  • High-contrast sharp shadows
✗ DON'T · Subject Matter
  • Busy, loud, or overly styled scenes
  • Group or corporate imagery
  • Stock-looking smiling faces
Image Text Overlay — Required CSS
background: linear-gradient(to top, rgba(46, 26, 26, 0.85) 0%, rgba(46, 26, 26, 0.00) 100% ); /* Always apply before placing any text over an image */

Tone of Voice

RARE speaks like a trusted, unhurried friend — never a salesperson. Every word should feel considered, warm, and earned.

Situation✓ On-Brand — Write Like This✗ Off-Brand — Never This
Product launch"A new ritual has arrived.""NEW DROP — shop now!"
Sale / offer"A quiet offer, just for you.""50% OFF THIS WEEKEND ONLY!!!"
Call to action"Begin the ritual →""Buy Now" / "Click Here"
Error message"Something feels off. Let's try again.""Error 404 — Page not found"
Empty cart"Your ritual awaits. Start here.""Your cart is empty!"
Confirmation"Your order is on its way to you.""Order confirmed! Yay!"
Newsletter signup"Join the ritual. Quietly.""Subscribe for EXCLUSIVE deals!"
Out of stock"This one is resting. Check back soon.""SOLD OUT"
The Golden Writing Rule

If it could appear in a loud advertisement, it doesn't belong in RARE. Calm your copy until it feels like a note left on a pillow.

13

Extended Never-Do List

  • Never use Champagne Gold as a background colour.
  • Never use more than one italic word per headline.
  • Never use harsh drop shadows — soft only (max 0.15 opacity).
  • Never use pure white (#FFFFFF) — always use Warm Cream (#FAF4EE).
  • Never use more than two accent colours on one screen.
  • Never set body copy in Dark Mocha — use Mauve (#7A4A4A).
  • Never use cool-toned or blue-grey photography.
  • Never place text on images without the gradient overlay.
  • Never use Gold as a focus outline — use Terracotta (#A4594A) for accessibility.
  • Never use letter-spacing on body copy — reserved for labels and buttons only.
  • Never use system fonts (Arial, Helvetica, Times New Roman).
  • Never use arbitrary spacing values — stay on the 8pt scale.
  • Never use animation durations over 0.6s — it feels heavy.
  • Never use bouncy or elastic animation easing.
  • Never underline text unless it is a hover or link state.
